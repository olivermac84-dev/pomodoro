# Pomodoro Timer

Modern Pomodoro timer built with React + Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy on Vercel

This project is configured for Vercel with `vercel.json`.

1. Push this repository to GitHub/GitLab/Bitbucket.
2. Import the project in Vercel.
3. Keep defaults (Framework: **Vite**, Build: `npm run build`, Output: `dist`).
4. Deploy.

For SPA routing support, all routes are rewritten to `index.html` via `vercel.json`.
